Primeiros passos com HTML


HTML, que significa HyperText Markup Language, é a linguagem padrão para criar páginas da web. Ele fornece uma estrutura básica para o conteúdo da web, indicando como o texto deve ser estruturado, quais imagens devem ser exibidas e como os links devem se comportar.

A estrutura básica de um documento HTML envolve o uso de várias tags que delimitam diferentes seções do conteúdo.


Construindo um código HTML básico


Para construir o código, é necessário que se faça o uso de tags. Em HTML, as "tags" são elementos fundamentais que permitem estruturar e formatar o conteúdo de uma página web.

As tags são cercadas por colchetes angulares "<tag>" para a abertura e "</tag>"com a inclusão de "/" para o fechamento. Cada tag tem uma função específica e contribui para a organização, definição de informações para o navegadpr e apresentação do conteúdo para o leitor.

Para inicio do código, é necessário informar ao navegador o tipo de docimento que será exibido. O "<!DOCTYPE html>" especifica que a página está usando HTML5, a versão mais recente e comum do HTML.

Para que este código seja melhor entendido, é necessário a presença de comentários nele. Os comentários aparecem entre "<!--" e "-->".


Iniciando...


<!DOCTYPE html> <!-- Comunicando ao navegador que este arquivo é do tipo HTML. -->

Todo o conteúdo HTML

Todo o conteúdo HTML vai estar dentro da tag 'html'. Nesta tag, o atributo "lang" vai determinar a linguagem do site (o padrão é o Inglês).

    <h1>Primeiros passos com HTML</h1> 
    <br>
    <p>HTML, que significa HyperText Markup Language, é a linguagem padrão para criar páginas da web. Ele fornece uma estrutura básica para o 
        conteúdo da web, indicando como o texto deve ser estruturado, quais imagens devem ser exibidas e como os links devem se comportar.</p>
    <p>A estrutura básica de um documento HTML envolve o uso de várias tags que delimitam diferentes seções do conteúdo.</p>
    
    <h2>Tags HTML Básicas</h2>
    <p>Existem várias tags HTML básicas que você pode usar para estruturar o conteúdo da sua página. Algumas delas incluem:</p>
    <ul>
        <li><code>&lt;h1&gt; a &lt;h6&gt;</code>: Cabeçalhos de diferentes níveis.</li>
        <li><code>&lt;p&gt;</code>: Parágrafos de texto.</li>
        <li><code>&lt;br&gt;</code>: Quebra de linha.</li>
        <li><code>&lt;ul&gt;</code> e <code>&lt;ol&gt;</code>: Listas não ordenadas e ordenadas.</li>
        <li><code>&lt;li&gt;</code>: Itens de lista.</li>
    </ul>

    <h2>Próximos Passos</h2>
    <p>Agora que você aprendeu os primeiros passos com HTML, você pode explorar mais recursos, como:</p>
    <ul>
        <li>Adição de imagens com a tag <code>&lt;img&gt;</code>.</li>
        <li>Criação de links com a tag <code>&lt;a&gt;</code>.</li>
        <li>Estilização usando folhas de estilo CSS.</li>
        <li>Integração de scripts com JavaScript.</li>
    </ul>
